# Modifier

## Adverb
- `Such a`
  - That was `such a` fun day.
  - You are `such a` pro. (まさしく)
- `All kinds of ...`
  - I've been eating `all kinds of things`.
- `Like`
  - There were only `like` thirty peeple there.
- `Kind of`
  - I'm `kind of` hungry.
  - I'm `kind of` sleepy.
  - She is `kind of` rude.
  - It was `kind of` ... confusing.
