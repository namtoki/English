# For the lesson

## 聞き取れなかった
- [x] **Could you** say that again?
- [ ] **Could you** repeat that, please?
- [ ] **Could you** speak a little slowly, please?
- [ ] **Could you** speak more slowly, please?
- [ ] **Could you** explain that?

## 内容が理解できなかった
- [ ] I'm sorry, I don't understand.
- [ ] Could you say that more simply, please?
- [ ] Could you write it down, please?
- [ ] Could you type it in the chat, please?

## 答えが思い浮かばない
- [ ] Let me see...
- [ ] I can't remember ...
- [ ] It's difficult to say.

## 質問
- [x] Can I ask you a question?
- [ ] I want to know about...
- [x] How do you pronounce this word?
- [x] How do you spell this word?
- [x] How do you say ... in English?
- [ ] What does ... mean?
- [ ] Is ... the same as ...?

## レッスンの終わりに
- [ ] Thank you for the lesson!
- [ ] Enjoy your ...
- [ ] Have a nice day!
- [ ] See you next time!
