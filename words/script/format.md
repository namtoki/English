# FORMAT

- `word` /pronunciation/        ([casualty] 訳)             例文

## 補足 1

- word:
  - format: 英単語を ` で挟んでください
  - position: `-` から 1 つスペースを空けて続けて置いてください
- pronunciation:
  - format: IPA 方式の発音記号を / で挟んでください
  - position: word から 1 つスペースを空けて続けて置いてください
- ():
  - format: calsualty と訳を () で挟んでください
    - casualty:
      - format: Casual, Formal, Neutral, Polite, Writing のいずれかを [] で挟んでください
        - Casual: 友人などと口語でよく用いられる
        - Formal: 会社の上司、取引先、初めて会う人と口語で用いられる
        - Neutral: Casual にも Formal にも口語で用いられる
        - Polite: 礼儀正しい口語で用いられる
        - Writing: 口語ではあまり用いられず、文章で主に用いられる
      - position: ( の直後、スペースを入れずに続けて置いてください
    - 訳:
      - format: word の訳を日本語で10文字以内で書いてください
      - position: casualty から 1 つスペースを空けて続けて置いてください
  - position: 行左端から数えて 37 文字目にしてください。pronunciation から ( までの間はスペースで埋めてください。
- 例文:
  - format: 口語でよく用いられる際の例文を一つ書いてください。casualty が Writing の場合は文章でよく用いられる例文で良いです。
  - position: 行左端から数えて 60 文字目にしてください。) から例文までの間はスペースで埋めてください。

## 例

- `hard` /hɑːd/                     ([Neutral] 努力を要する難しさ)  The test was hard.
