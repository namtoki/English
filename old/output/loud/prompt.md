# 要件

これから入力する文章の発音記号を出力してください。

出力記号の形式は `../../cheat/1_base/0_pronunciation.md` の `## アメリカ英語の母音一覧表` 中の補助記号を使用してください。
（IPA 形式から補助記号形式への変換については、同表中に記載しています）

出力の最初には `[/` 最後には `/]` を必ずをつけてください。

また、文章中の単語それぞれ個別に発音記号を生成するのではなく、文として発話された際のリンキング、Flap T、弱系変化 (機能語は必ず弱形にし、文中の重要でない単語も弱形にする。弱形とは母音がほぼ uh になることを言う) を
考慮するようにお願いします。

発音変化に関して、ルールの一部を `../../cheat/1_base/0_pronunciation.md` の `## 応用` 以下の行に記載しているので、参考にしてください。

アメリカ英語の発音でお願いします。

最後に、上記ポイントの後に改行し、別の言い回しでより頻繁に使用される表現、熟語、単語などあれば 1 つ出力してください (動詞表現に着目するのを優先してください)。
その出力に対する発音記号も、改行後に出力してください。
発音記号の後に改行し、なぜその表現の方が頻繁に使用されるのか、元々の入力時の表現だとどういったニュアンスがあるのか (少しフォーマルよりだ、とか) 出力してください。
