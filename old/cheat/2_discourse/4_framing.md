- [x] `from one's perspective`              ([Neutral] ～の視点からすると) `From my perspective,` the plan needs some adjustments.
- [x] `from one's point of view`            ([Neutral] ～の視点から見れば) `From my point of view,` the decision makes sense.
- [x] `from the perspective of O`           ([Neutral] Oの観点から見ると) `From the perspective of` the customer, this service is excellent.

## 前の文を条件として
- [x] `that's how`                          ([Casual] このようにして) `That's how` we met.
- [x] `(in) this/that way`                  ([Neutral] このようにして) We should ~~ `in this way` to ~~. / Leave early. / `That way,` you won’t be late.
- [x] `in which case`                       ([Neutral] その場合は) We could cancel the event, `in which case` we need to notify everyone.
- [x] `in that case`                        ([Neutral] その場合は) If you're busy, `in that case,` we can reschedule.
- [ ] `by doing this`                       ([Neutral] そうすることによって) You’ll improve your skills `by doing this regularly.`
- [ ] `and then`                            ([Neutral] それから) `And then,` we decided to go home.
- [ ] `then`                                ([Neutral] そうすれば) Do whatever I say. `Then` you will be successful.
- [ ] `thus`                                ([Formal] [文語] そうすれば) Mix the ingredients together, `thus` creating a smooth paste.
